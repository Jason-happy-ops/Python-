Day 15 2025.12.4
我决定好好理解一下这个可变&不可变类型：不可变数据类型： 当该数据类型的对应变量的值发生了改变，那么它对应的内存地址也会发生改变，对于这种数据类型，就称不可变数据类型。
                                    可变数据类型 ：当该数据类型的对应变量的值发生了改变，那么它对应的内存地址不发生改变，对于这种数据类型，就称可变数据类型。
之前一直不理解为什么有了列表还要有元组，现在从内存优化以及安全方面好像确实很有道理。
