Day 25 2025.12.27
今天完成的是模拟退火算法的初步探究。什么是模拟退火算法呢？先设定一个随机起点X，以及初始温度T。算法开始后，x随机向左或右移动一小格，相应的温度也要下降一些。若移动后的y2<初始y1
那么y2对应的x2成为新的x，继续这一个过程，直到温度T小于某一个特定值（阈值）。但是需要注意的是，模拟退火算法允许更差的解，但是需要概率辅助计算：
accept_prob = math.exp(-E_minus / T)。这有利于我们跳出局部最优解法去寻找全局最优解法。
